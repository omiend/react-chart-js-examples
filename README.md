# react-chart-js-examples

## clone

```
$ git clone git@github.com:omiend/react-chart-js-examples.git
$ cd react-chart-js-examples
```

## npm install
```
$ npm install
```

## start
```
$ npm start
```

go to [http://localhost:3000/](http://localhost:3000/)



## compile
```
$ webpack --progress --color --watch
```
